# Nudge Legal Site

Static legal and support pages for Nudge.

## Pages

- `index.html` – Landing page
- `privacy.html` – Privacy Policy
- `terms.html` – Terms of Service
- `support.html` – Support page
- `delete-account.html` – Account and data deletion instructions

## GitHub Pages

Upload all files to the repository root and enable GitHub Pages.

Expected structure:

```text
Nudge-legal/
├── index.html
├── privacy.html
├── terms.html
├── support.html
├── delete-account.html
├── style.css
└── assets/
    └── logo.png
```
